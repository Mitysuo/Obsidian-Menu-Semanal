# 🍽️ Obsidian Menu Semanal

Automações para planejar o cardápio semanal de refeições diretamente pela interface de notas do [Obsidian](https://obsidian.md/).

O projeto gera dashboards e relatórios de acompanhamento de macronutrientes, além de montar a lista de compras do supermercado com base nos ingredientes necessários para a semana.

Os dados de macronutrientes são obtidos da **TBCA** (Tabela Brasileira de Composição de Alimentos).

## ✨ Funcionalidades

- Planejamento do cardápio semanal via notas do Obsidian
- Cálculo e acompanhamento de macronutrientes (carboidratos, proteínas, gorduras)
- Geração automática da lista de compras
- Dashboards de acompanhamento nutricional

## 💾 Pré-requisitos

- **Python** 3.12
- **Poetry** 2.4.1

## 🚀 Como usar

```bash
poetry lock
poetry install --with test
```
